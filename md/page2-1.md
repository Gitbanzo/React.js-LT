
## 2.環境構築
--------

*今回はversion 14.0.6 (2016/04/22現在最新 v15.0.1)*

### 2.1 React.jsのインストール

#### 2.1.1 npmインストールする

```
$ npm isntall --save react@0.14.6
$ npm isntall --save react-dom@0.14.6
```

#### 2.1.2 Yeomanから作成する

```
$ npm i -g yo
$ npm i -g generator-react-6
$ yo react-6
```
[generator-react-6](https://www.npmjs.com/package/generator-react-6)
